# c-cowsay : Cowsay In C Lang

```bash
# Build
$ gcc cowsay.c -o cowsay

# Exec
$ ./cowsay Message
  _______
< Message >
  -------
     \  ^__^
      \ (oo)\_____
        (__)\     )\/\
           ||----w |
           ||     ||

# No Arguments
$ ./cowsay
Usage : cowsay [Message]
```
